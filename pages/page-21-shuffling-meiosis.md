## Goal of This Page

Meiosis — the division that creates sex cells. TWO concepts: (1) Meiosis makes gametes (sperm and eggs) with HALF the chromosomes — 46 → 23, so that sperm (23) + egg (23) = embryo (46), and (2) Independent assortment — each of the 23 chromosome pairs is randomly assigned to each gamete, giving 2^23 = ~8.4 million possible combinations, even BEFORE recombination. Side-by-side with mitosis to make the contrast visceral.

🟦 Sperm (23 chromosomes) + 🟠 Egg (23 chromosomes) = 🟢 Embryo (46 chromosomes)

Helix: "Why half? Because when sperm meets egg, they COMBINE. 23 + 23 = 46. If they didn't halve first, every generation would DOUBLE the chromosome count."

### Slots 5-6 (bottom-left, merged 2 slots) — "The Shuffle: Independent Assortment"

Wide panel. A visual of 23 chromosome PAIRS lined up in a row (represented as paired colored bars — one from mom in orange, one from dad in blue). Below them, a "coin flip" happening at each pair — heads means the mom version goes into the gamete, tails means the dad version.

The result: a row of 23 chromosomes, each randomly chosen from the pair. Some orange, some blue, in a random pattern.

Helix: "You have 23 PAIRS — one from your mom, one from your dad. When making a gamete, each pair independently flips a coin. Mom's version or dad's? Multiply that across all 23 pairs..."

Handwritten (large, emphasized): "2²³ = 8,388,608 possible combinations. From ONE person. Before anything else happens."
